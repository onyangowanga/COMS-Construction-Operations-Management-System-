# Logo Directory

Place your logo files here:

## Required File:
- **logo.png** - Your main logo (recommended size: 300x80px or similar aspect ratio)
  - Used in: Navbar and login page
  - Background: Transparent PNG recommended
  - Color: White or light colors work best (navbar has dark background)

## Optional Files:
- **logo-dark.png** - Dark version of logo (for light backgrounds)
- **logo-full.png** - Full size logo for reports/documents

## Usage:
The logo will automatically appear in:
- Navigation bar (height: 30px)
- Login page (max-height: 80px)
- Registration page

## Tips:
- Use PNG format with transparent background
- Keep file size under 200KB
- SVG format also supported for scalability

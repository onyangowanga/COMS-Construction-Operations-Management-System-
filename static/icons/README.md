# Favicon & App Icons Directory

Place your favicon and app icon files here:

## Required Files:

### Favicon (Browser Tab Icon):
- **favicon.ico** - Standard favicon (32x32 or 16x16)
- **favicon-16x16.png** - 16x16 PNG favicon
- **favicon-32x32.png** - 32x32 PNG favicon

### Mobile/PWA Icons:
- **apple-touch-icon.png** - 180x180 PNG for iOS devices
- **android-chrome-192x192.png** - 192x192 PNG for Android
- **android-chrome-512x512.png** - 512x512 PNG for Android

## How to Create Favicons:

### Option 1: Online Generator
1. Go to https://realfavicongenerator.net/
2. Upload your logo/icon
3. Download the generated package
4. Extract all files to this folder

### Option 2: Manual Creation
Using your logo image:
1. Create square versions (512x512px recommended)
2. Resize to required sizes (16x16, 32x32, 180x180, 192x192, 512x512)
3. Save as PNG files with specified names
4. Convert 32x32 PNG to .ico format for favicon.ico

## Quick Start:
If you only have one logo/icon:
1. Name it `favicon.ico` or `favicon-32x32.png`
2. The browser will use it automatically
3. Add others later for better mobile experience
